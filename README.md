# Cbuf

This is a fork repository of [ocaml-ctypes](https://github.com/ocamllabs/ocaml-ctypes)

## Getting Stared

Cbuf のインストール

```sh
$ make && make install
```

NOTE: Ctypes がすでにインストールされている場合は、アンインストールしてからインストールしてください。

```sh
$ make uninstall
```
